# Private
Private privacy preserving analysis language

## Installation
Use python setuptools:
```bash
python setup.py install
```

## Usage
To open an interactive session
```bash
cd Private
python command.py
```

To run commands from a file
```bash
python command.py yourCommandsFile.pr
```
